# Physical Site Analysis
___
Date: <% tp.date.now('YYYY-MM-DD') %>
Assessor: 
Site: 
Review Type: `Annual / Biannual / Monthly`
Stakeholders Present:
>

## Cameras
___
- [ ] All cameras working and covering each gate and building door
- [ ] Camera connection types identified (coaxial, Ethernet, wireless)
- [ ] Ethernet cameras checked for unpluggable ports that provide network access
- [ ] Wireless cameras confirmed on a secured network, not the process network
- [ ] Camera recordings can be exported and saved

Notes:
>

## Perimeter
___
- [ ] Fence line walked, all gates confirmed locked
- [ ] Checked for low points where erosion allows movement under the fence
- [ ] Checked for exterior objects (trees, walls, equipment) that aid climbing over
- [ ] Gate exit mechanisms tested to confirm they cannot be used to unlock from outside

Notes:
>

## External Cabinets
___
⚠️ Watch for bee/wasp nests when opening cabinets.

- [ ] Each external control cabinet confirmed locked
- [ ] Cabinets reviewed for tamper tape or sensors indicating door opened
- [ ] Cabinets reviewed for network/process documentation (should not be present)
- [ ] Device physical security mechanisms verified enabled (e.g., PLC key switches)

Notes:
>

## Building Access
___
- [ ] Each building door confirmed to lock properly
- [ ] Door frames checked for exposed throws and bypass vulnerability (shifted foundations, under/over-door tools)
- [ ] Doors checked for tamper tape or sensors indicating door opened
- [ ] Buildings checked for ladders leading to roof access
- [ ] Roof access points (doors, hatches) confirmed locked with tamper tape or sensors

Notes:
>

## Internal Cabinets and Network Racks
___
- [ ] Control cabinets, network racks, and server racks have doors installed, locks available and in use
- [ ] Tamper tape or sensors in use to indicate when cabinet/rack doors have been opened
- [ ] Critical cabinets or closets monitored by cameras
- [ ] Regular physical walkthroughs conducted for unauthorized devices and network connections

Notes:
>

## General
___
- [ ] Buildings reviewed for unexpected wireless antennas (e.g., cellular modems on generators for vendor maintenance)
- [ ] Unauthorized access checklists reviewed to confirm cybersecurity team is notified of physical security events

Notes:
>
