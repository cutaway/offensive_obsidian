## Change Categories Reference
___

- Account changes - adding, modifying, or disabling user accounts or credentials
- File uploads - scripts, tools, payloads, binaries placed on target systems
- Configuration changes - firewall rules, registry keys, service settings, PLC logic, HMI configurations
- Feature or service changes - enabling, disabling, or modifying services or protocols
- Software installations - agents, implants, services, utilities
- Database modifications - inserts, updates, deletes to application or historian databases
- Persistence mechanisms - scheduled tasks, cron jobs, startup scripts, registry run keys
- Network changes - routes, DNS entries, VLAN assignments
- Logging or monitoring changes - disabling or modifying audit policies, log forwarding, SIEM rules
